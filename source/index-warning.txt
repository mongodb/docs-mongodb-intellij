Index Performance Warnings

Definition
----------

The {+intellij-full+} examines if application queries use indexes. If a
query doesn't use an index or is only partially covered by an index, the plugin 
displays a warning for that query.

To resolve the warning, consider creating an index for the query.

Before you add an index, consider if:

- The query runs often enough to justify reducing the write performance
  for faster reads.
- You can change the query to use an existing index.

You can also disable index warnings.

For more information about indexes, see :ref:`indexes`.

Examples
--------

In the following example Java code snippet, the ``awards`` document
field is used in a query, but the field isn't indexed in the database:

.. code-block:: java
   :copyable: false
   :emphasize-lines: 2

   client.getDatabase( "sample_mflix" ).getCollection( "movies" ).find(
      Filters.ne( "awards", "Comedy" )
   )

The side panel shows the following warning under 
:guilabel:`Performance Warnings`:

.. image:: /images/index-warning.png
   :alt: Index warning in the IntelliJ Plugin.
   :width: 400px

Create an Index
~~~~~~~~~~~~~~~

To create an index for the query, click the :guilabel:`Create an
Index` button displayed in the side panel warning.

The plugin then displays the :guilabel:`Database Explorer Playgrounds`
screen with template code for creating an index:

.. code-block:: java
   :copyable: false
   :emphasize-lines: 1

   // region Queries covered by this index 
   // alt.mongodb.javadriver.JavaDriverRepository#getRatings at line 32
   // endregion 
   // Learn about creating an index: https://www.mongodb.com/docs/v7.0/core/data-model-operations/#indexes
   db.getSiblingDB("sample_mflix").getCollection("movies")
      .createIndex({ "awards": 1 })

To create an index for the ``awards`` field, set ``<your_field_1>``
to ``awards`` in the example code and then run the
:method:`~db.collection.createIndex()` method in the :guilabel:`Database
Explorer Playgrounds` screen. For example:

.. code-block:: java
   :emphasize-lines: 2

   db.getSiblingDB("sample_database").getCollection("movies").
   createIndex({"awards": 1})

Disable Index Warning
~~~~~~~~~~~~~~~~~~~~~

To disable the index warning in the side panel:

.. procedure:: 
   :style: normal
   
   .. step:: Click the :icon-lg:`VerticalEllipsis` icon next to the warning.
      
   .. step:: Click :guilabel:`Disable Inspection`.

      When you click :guilabel:`Disable Inspection`, the plugin disables all 
      index warnings for the project.

If you previously disabled index warnings and want to re-enable the insight: 

.. procedure:: 
   :style: normal

   .. step:: Under :guilabel:`Performance Warnings`, click :guilabel:`Show disabled inspections`.
      
   .. step:: Click the :icon-lg:`VerticalEllipsis` icon next to the hidden warning.

   .. step:: Click :guilabel:`Enable Inspection`.
